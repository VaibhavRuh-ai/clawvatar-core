Use with Platforms

Zoom / Google Meet / Teams (Tab Sharing)

  1. Set audio input to "Microphone" above
  2. Open the Stream View URL in a new tab
  3. Grant mic access — agent hears you through LiveKit
  4. In your meeting: Share Screen → Chrome Tab
  5. Check "Share tab audio" so others hear the agent

Zoom / Meet (Audio Bridge)

  1. Set audio input to "System audio" above
  2. Open the Stream View URL in a new tab
  3. When prompted, select the meeting tab to capture its audio
  4. Meeting participants' voices → agent hears them
  5. Share the avatar tab for video output

Direct LiveKit Room (Simplest)

  1. Share the Stream View URL directly with participants
  2. Everyone opens it — joins the same LiveKit room
  3. No Zoom/Meet needed — full bidirectional voice

OBS Studio

  1. Add a Browser Source → paste Stream View URL
  2. Set size to match resolution, enable audio
  3. Use OBS Virtual Camera → select in Zoom/Meet

YouTube Live / Twitch

  1. Enable RTMP output, paste your stream key
  2. Start stream — video+audio goes directly to platform

Embed on Website (Recommended)

  1. Pick an agent above, copy the embed code below
  2. Paste into your website HTML
  3. Visitors click mic to talk to the avatar
